This repository contains everything that is necessary to replicate
the construction of the database and the results presented in the figures
and table of the draft of the article Arboleda, Purcell and Roblero (2021).

"Final Databases" folder contain the final datasets: 'data_world_regions'
and 'data_sample' (you will find them in DTA and XLS)

"Replication Codes" folder contains the STATA do-files required to construct
the final datasets ('Replication Code - Databases Construction') and to replicate
data processing and analyzes, and figures and table construction ('Replication Code - Analysis and Results')

"Final Figures and Table" folder contains figures and table which should be obtained when running the code
(some figures and the table have to be edited in excel)

"Data Sources Folder" contains original dataset that were used to create de final databases.
They came from:

Food-related GHG emissions data:
1. Edgar-Food: https://edgar.jrc.ec.europa.eu/edgar_food#sources (Downloaded on December 12, 2021)

Political-economic data:

4. Exchange Rates: https://data.worldbank.org/indicator/PA.NUS.FCRF (Downloaded on December 13, 2021)
5. Nominal GDP: https://data.worldbank.org/indicator/NY.GDP.MKTP.CD (Downloaded on December 13, 2021)
6. GDP deflator: https://data.worldbank.org/indicator/NY.GDP.DEFL.KD.ZG?locations=UShttp%3A%2F%2Fdata.worldbank.org%2Findicateur%2FNE.EXP.GNFS.CD%3Flocations%3DUS (Downloaded on December 13, 2021)

8. CPI Argentina: https://fred.stlouisfed.org/series/DDOE02ARA086NWDB (Downloaded on December 13, 2021)
